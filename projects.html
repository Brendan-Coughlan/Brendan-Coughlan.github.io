Interview Coach

Interview Coach is an AI-powered web application developed by a team of three for CSC 430: Software Engineering. The platform helps job seekers prepare for interviews by offering personalized mock sessions, resume-based question generation, and instant AI feedback. Built with Next.js, FastAPI, and the OpenAI API, it allows users to practice industry-specific questions, refine their responses, and track progress over time. This project showcases our ability to design and deliver a real-world solution using modern technologies and agile development practices.

KotlinTrade

Kotlin Trade is a mobile application developed by a team of three for CSC 438: Mobile Application Development. The app helps users track their stock portfolios by offering real-time price updates, customizable alerts, and a clean, intuitive interface. Built with Kotlin and integrated with the Polygon.io API, it allows users to search for stocks, monitor performance, and stay informed through background notifications. This project demonstrates our ability to design and deliver a functional, user-focused solution using modern development practices and collaborative tools like GitHub and wireframing.
